To-Do List Application
A simple and interactive To-Do List web application built with HTML, CSS, and JavaScript. This application allows users to manage their daily tasks, mark them as complete, delete completed items, and even restore previously deleted tasks. It also features a dark mode toggle for comfortable viewing.

Deployment Link: https://todolist-js-sandy.vercel.app/

Features
Add New Tasks: Easily add new items to your to-do list using the input field.

Mark as Complete: Click on any task to toggle its completion status (adds a strikethrough).

Mark Last Complete: A dedicated button to quickly mark the last added task as complete.

Delete Completed Tasks: Remove all tasks that have been marked as complete.

Reset All: Clear all completion marks and restore any previously deleted tasks.

Dark Mode Toggle: Switch between light and dark themes for improved readability and user preference.

Responsive Design: Adapts to various screen sizes for a seamless experience on desktop and mobile devices.

Sidebar Navigation: A collapsible sidebar for future integration of other productivity tools.

UI and Interaction
The application is designed with a clean and intuitive user interface to ensure ease of use.

Visual Clarity: Tasks are presented in a clear list format, with distinct visual cues for completion (strikethrough and faded text).

Interactive Elements: Buttons are clearly labeled with icons and text, providing immediate feedback on their actions.

Theme Switching: The dark mode feature offers a personalized viewing experience, reducing eye strain in low-light environments.

Fluid Layout: The responsive design ensures that the layout adjusts gracefully across different screen sizes, from mobile phones to desktop monitors, maintaining usability and aesthetic appeal.

Smooth Transitions: CSS transitions are used to provide smooth visual feedback on interactions, such as hovering over list items or toggling the theme.

Technologies Used
HTML5: For the structure of the web application.

CSS3: For styling and responsive design, including theme switching.

JavaScript: For all interactive functionalities, task management, and DOM manipulation.
